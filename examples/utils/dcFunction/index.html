DCJS - dcFunction 工具类示例

dcFunction 是 DCJS 库中的函数工具类，提供了一系列常用的函数处理方法，如防抖、节流、柯里化、组合等。本示例页面展示了所有功能的用法。

1. 防抖函数 (debounce)

防抖函数用于限制函数在短时间内的多次调用，只有当一段时间内没有再调用时，才会执行最后一次调用。

示例：输入框搜索防抖

搜索结果： 等待输入...
实际搜索次数： 0
说明： 当你停止输入 500ms 后，才会执行搜索操作，减少不必要的请求。

2. 节流函数 (throttle)

节流函数用于限制函数的执行频率，确保在一段时间内只执行一次。

示例：滚动事件节流

向下滚动此区域...

滚动位置：0

节流后执行次数： 0
说明： 滚动事件会被节流，每 200ms 只执行一次，提高性能。

3. 柯里化函数 (curry)

柯里化函数将一个多参数函数转换为一系列单参数函数。

// 原始函数 function add(a, b, c) { return a + b + c; } // 柯里化后 const curriedAdd = DC.utils.dcFunction.curry(add); // 用法 const add5 = curriedAdd(5); const add5And10 = add5(10); const result = add5And10(15); // 30
结果： 点击按钮测试

4. 函数组合 (compose)

函数组合将多个函数组合成一个新函数，从右到左执行。

// 定义多个函数 function add1(x) { return x + 1; } function multiply2(x) { return x * 2; } function square(x) { return x * x; } // 组合函数 const composedFn = DC.utils.dcFunction.compose(square, multiply2, add1); // 执行：先 add1，再 multiply2，最后 square const result = composedFn(3); // ((3 + 1) * 2)² = 64
结果： 点击按钮测试

5. 管道函数 (pipe)

管道函数将多个函数组合成一个新函数，从左到右执行。

// 定义多个函数 function add1(x) { return x + 1; } function multiply2(x) { return x * 2; } function square(x) { return x * x; } // 管道函数 const pipedFn = DC.utils.dcFunction.pipe(add1, multiply2, square); // 执行：先 add1，再 multiply2，最后 square const result = pipedFn(3); // ((3 + 1) * 2)² = 64
结果： 点击按钮测试

6. 记忆化函数 (memoize)

记忆化函数用于缓存函数的计算结果，避免重复计算。

// 定义一个耗时的函数 function fibonacci(n) { if (n <= 1) return n; return fibonacci(n - 1) + fibonacci(n - 2); } // 记忆化处理 const memoizedFib = DC.utils.dcFunction.memoize(fibonacci); // 第一次调用会计算 const result1 = memoizedFib(30); // 第二次调用会直接返回缓存的结果 const result2 = memoizedFib(30);
结果： 点击按钮测试

7. 偏函数 (partial)

偏函数用于固定函数的部分参数，返回一个新函数。

// 原始函数 function greet(greeting, name) { return `${greeting}, ${name}!`; } // 创建偏函数，固定第一个参数 const sayHello = DC.utils.dcFunction.partial(greet, 'Hello'); // 调用偏函数 const result = sayHello('World'); // Hello, World!
结果： 点击按钮测试

8. 函数重载 (overload)

函数重载允许一个函数根据参数类型和数量有不同的行为。

// 创建重载函数 const overloadedFn = DC.utils.dcFunction.overload( [String], (str) => `Hello, ${str}!`, [Number], (num) => num * 2, [String, Number], (str, num) => `${str} repeated ${num} times` ); // 调用不同版本 const result1 = overloadedFn('World'); // Hello, World! const result2 = overloadedFn(5); // 10 const result3 = overloadedFn('Hi', 3); // Hi repeated 3 times
