dcHttps 网络请求工具示例

工具介绍

dcHttps 是一个简化网络请求的工具类，提供了以下功能：

  • get(url) - 发起 GET 请求
  • post(url, data) - 发起 POST 请求

注意：由于浏览器的同源策略，直接在本地打开此页面可能无法正常发起跨域请求。本示例使用 JSONPlaceholder 提供的模拟 API 进行演示。
