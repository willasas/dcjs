dcNetworkChecker 网络连接检测工具类示例

1. 网络连接测试

测试当前网络连接状态。

点击按钮开始测试
// 测试网络连接 networkChecker.testConnection() .then(result => { console.log('网络连接测试结果:', result); }) .catch(error => { console.error('测试网络连接时出错:', error); }); // 带重试的网络连接测试 networkChecker.testConnectionWithRetry() .then(result => { console.log('带重试的网络连接测试结果:', result); }) .catch(error => { console.error('测试网络连接时出错:', error); });

2. 网络信息获取

获取当前网络的详细信息。

// 获取网络信息 const networkInfo = networkChecker.getNetworkInfo(); console.log('网络信息:', networkInfo); console.log('网络类型:', networkInfo.effectiveType); console.log('下行带宽:', networkInfo.downlink, 'Mbps');

3. 网络状态监控

实时监控网络状态变化。

监控状态: 未启动
点击按钮开始监控
// 开始监控网络状态 networkChecker.startMonitoring(); console.log('网络监控已启动'); // 停止监控网络状态 networkChecker.stopMonitoring(); console.log('网络监控已停止');

4. 网络状态摘要

获取当前网络状态的完整摘要。

点击按钮开始测试
// 获取网络状态摘要 networkChecker.getStatusSummary() .then(summary => { console.log('网络状态摘要:', summary); console.log('最后检查时间:', summary.lastChecked); }) .catch(error => { console.error('获取网络状态摘要时出错:', error); });

根据网络质量调整应用行为。

点击按钮开始测试
// 检测网络质量 async function detectNetworkQuality() { try { const networkInfo = networkChecker.getNetworkInfo(); const connectionResult = await networkChecker.testConnection(); console.log('网络质量检测结果:', { networkInfo, connectionResult }); // 根据网络类型调整应用行为 if (networkInfo.effectiveType === '4g') { console.log('网络良好，加载高清内容'); } else if (networkInfo.effectiveType === '3g') { console.log('网络一般，加载中清内容'); } else { console.log('网络较差，加载低清内容'); } } catch (error) { console.error('检测网络质量时出错:', error); } }
