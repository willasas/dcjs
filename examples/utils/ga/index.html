ga 工具类示例

注意：本示例使用的是 Google Analytics 经典版（ga.js），该版本已于 2023 年 7 月 1 日停止服务。建议使用 Google Analytics 4（gtag.js）替代。

工具类介绍

ga 工具类提供了一个简单的函数 initGoogleAnalytics，用于初始化 Google Analytics 经典版跟踪代码。

主要功能：

  • 创建并初始化 _gaq 全局数组
  • 设置 Google Analytics 账户 ID
  • 发送页面浏览事件
  • 动态加载 Google Analytics 跟踪脚本
  • 根据当前协议（http/https）选择合适的脚本 URL

使用示例

// 引入 ga 工具类
import initGoogleAnalytics from 'path/to/ga.js';

// 初始化 Google Analytics
// 注意：请替换为您自己的 Google Analytics 账户 ID
initGoogleAnalytics('UA-12345678-1');

// 发送自定义事件
try {
  if (window._gaq) {
    window._gaq.push(['_trackEvent', 'Category', 'Action', 'Label']);
  }
} catch (error) {
  console.error('发送事件失败:', error);
}

迁移到 Google Analytics 4

由于 Google Analytics 经典版已停止服务，建议迁移到 Google Analytics 4。以下是 GA4 的初始化代码示例：

// Google Analytics 4 初始化代码
function initGoogleAnalytics4(measurementId) {
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', measurementId);
  
  const script = document.createElement('script');
  script.src = `https://www.googletagmanager.com/gtag/js?id=${measurementId}`;
  script.async = true;
  const s = document.getElementsByTagName('script')[0];
  s.parentNode.insertBefore(script, s);
}

// 使用示例
initGoogleAnalytics4('G-XXXXXXXXXX');
