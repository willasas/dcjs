DCJS - JSON 工具类示例

什么是 DCJson？

DCJson 是一个 JSON 工具类，提供了一系列用于处理 JSON 数据的静态方法，包括：

  • 验证 JSON 字符串格式是否有效
  • 将 JSON 对象数组转换为 CSV 格式
  • 合并多个 JSON 对象
  • 深拷贝 JSON 对象

使用示例

示例 1：验证 JSON 格式

使用 isValidJSON 方法验证 JSON 字符串格式是否有效：

// 验证有效的 JSON 字符串 const validJson = '{"name": "John", "age": 30}'; const isValid = DC.Json.isValidJSON(validJson); console.log('Is valid JSON:', isValid); // 验证无效的 JSON 字符串 const invalidJson = '{"name": "John", "age": 30'; // 缺少闭合括号 const isInvalid = DC.Json.isValidJSON(invalidJson); console.log('Is invalid JSON:', isInvalid);

示例 2：JSON 转 CSV

使用 jsonToCsv 方法将 JSON 对象数组转换为 CSV 格式：

// 准备数据 const data = [ { name: 'John', age: 30, city: 'New York' }, { name: 'Jane', age: 25, city: 'London' }, { name: 'Bob', age: 35, city: 'Paris' } ]; // 定义列名 const columns = ['name', 'age', 'city']; // 转换为 CSV const csv = DC.Json.jsonToCsv(data, columns); console.log('CSV output:', csv); // 使用自定义分隔符 const csvSemicolon = DC.Json.jsonToCsv(data, columns, ';'); console.log('CSV with semicolon:', csvSemicolon);

示例 3：合并 JSON 对象

使用 mergeJsonObjects 方法合并多个 JSON 对象：

// 定义要合并的对象 const obj1 = { name: 'John', age: 30 }; const obj2 = { city: 'New York', country: 'USA' }; const obj3 = { job: 'Engineer', salary: 100000 }; // 合并对象 const merged = DC.Json.mergeJsonObjects(obj1, obj2, obj3); console.log('Merged object:', merged); // 处理属性冲突 const objA = { name: 'John', age: 30, city: 'Boston' }; const objB = { city: 'New York', country: 'USA' }; const mergedWithConflict = DC.Json.mergeJsonObjects(objA, objB); console.log('Merged with conflict:', mergedWithConflict);

示例 4：深拷贝 JSON 对象

使用 deepCopyJson 方法深拷贝 JSON 对象：

// 定义原始对象 const original = { name: 'John', age: 30, address: { street: '123 Main St', city: 'New York', zip: 10001 }, hobbies: ['reading', 'gaming', 'coding'] }; // 深拷贝对象 const copy = DC.Json.deepCopyJson(original); console.log('Original object:', original); console.log('Copied object:', copy); console.log('Are they the same reference?', original === copy); console.log('Are addresses the same reference?', original.address === copy.address); // 修改原始对象的嵌套属性 original.address.city = 'Boston'; console.log('After modifying original:', original.address.city); console.log('Copy remains unchanged:', copy.address.city);

使用技巧

1. 错误处理

在使用 jsonToCsv 方法时，需要注意以下几点：

  • 第一个参数必须是数组
  • 第二个参数必须是数组且不能为空
  • 第三个参数（如果提供）必须是字符串
// 正确用法 try { const csv = DC.Json.jsonToCsv(data, columns); console.log('CSV:', csv); } catch (error) { console.error('Error:', error.message); } // 错误用法（会抛出异常） try { const csv = DC.Json.jsonToCsv('not an array', columns); } catch (error) { console.error('Error:', error.message); }

2. 性能考虑

使用 deepCopyJson 方法时，对于大型 JSON 对象可能会影响性能，因为它使用了 JSON.parse(JSON.stringify()) 方法。如果只需要浅拷贝，可以使用 Object.assign 或扩展运算符。

3. 自定义分隔符

在使用 jsonToCsv 方法时，可以根据需要指定自定义分隔符，例如使用分号 ; 或制表符 \t：

// 使用分号作为分隔符 const csvSemicolon = DC.Json.jsonToCsv(data, columns, ';'); // 使用制表符作为分隔符 const csvTab = DC.Json.jsonToCsv(data, columns, '\t');

支持的浏览器

  • Firefox 3.5+
  • Safari 4+

注意：DCJson 工具类使用了原生的 JSON.parse 和 JSON.stringify 方法，这些方法在 IE 8 及以下版本中不支持。如果需要在这些浏览器中使用，建议添加 JSON polyfill。

常见问题

Q: 为什么深拷贝方法对某些值类型不适用？

A: deepCopyJson 方法使用 JSON.parse(JSON.stringify()) 实现，因此它不能处理以下类型的值：

  • 函数
  • 正则表达式
  • Date 对象（会被转换为字符串）
  • undefined
  • Symbol
  • 循环引用

Q: 如何处理 CSV 中的特殊字符？

A: jsonToCsv 方法会自动处理以下情况：

  • 在值周围添加双引号
  • 将值中的双引号替换为两个双引号（转义）
  • 处理 null 和 undefined 值，转换为空字符串
