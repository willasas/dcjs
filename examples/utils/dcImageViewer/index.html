DCImageViewer 示例

基本介绍

DCImageViewer 是一个功能强大的图片预览工具类，类似于 viewerjs，支持以下功能：

  • 图片预览和画廊功能
  • 图片放大、缩小功能
  • 图片旋转（左/右）
  • 图片水平和垂直翻转
  • 图片下载功能
  • 键盘快捷键支持
  • 响应式设计
  • 自定义配置选项
  • 平滑动画效果

示例 1: 单张图片预览

点击下方图片查看预览效果：

// 单张图片预览 const singleViewer = new DC.ImageViewer({ container: '#single-image-container' });

示例 2: 多图片画廊

点击下方任意图片查看画廊效果：

// 多图片画廊 const galleryViewer = new DC.ImageViewer({ container: '#gallery-container', navbar: true, toolbar: true, title: true });

示例 3: 自定义配置

带有自定义配置的图片预览：

// 自定义配置 const customViewer = new DC.ImageViewer({ container: '#custom-container', modal: true, backdrop: 'static', // 点击背景不关闭 keyboard: true, initialZoomRatio: 1, minZoomRatio: 0.1, maxZoomRatio: 5, navbar: false, // 隐藏导航栏 toolbar: true, title: true, button: true, animated: true, // 回调函数 viewed: function() { console.log('图片已查看'); }, shown: function() { console.log('查看器已显示'); }, hidden: function() { console.log('查看器已隐藏'); }, rotated: function() { console.log('图片已旋转'); }, zoomed: function() { console.log('图片已缩放'); } });

键盘快捷键

Esc: 关闭查看器 Arrow Left: 上一张图片 Arrow Right: 下一张图片 + 或 =: 放大图片 -: 缩小图片 0: 重置图片 r: 向右旋转 l: 向左旋转 f: 水平翻转

API 方法

// 打开查看器 viewer.open(index); // index 为图片索引 // 关闭查看器 viewer.close(); // 放大图片 viewer.zoomIn(); // 缩小图片 viewer.zoomOut(); // 向左旋转 viewer.rotateLeft(); // 向右旋转 viewer.rotateRight(); // 水平翻转 viewer.flipHorizontal(); // 垂直翻转 viewer.flipVertical(); // 重置 viewer.reset(); // 下载图片 viewer.download(); // 销毁查看器 viewer.destroy();
