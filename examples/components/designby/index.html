DCDesignBy Component Example

Basic Usage

This example demonstrates the basic usage of the DCDesignBy component.

// Sample data for the designby component
const designData = [
    {
        name: 'UI Design',
        icon: ''
    },
    {
        name: 'Web Development',
        icon: ''
    },
    {
        name: 'Mobile Apps',
        icon: ''
    },
    {
        name: 'Graphic Design',
        icon: ''
    },
    {
        name: 'Branding',
        icon: ''
    },
    {
        name: 'Animation',
        icon: ''
    }
];

// Initialize the designby component
const designBy = new DC.DesignBy('#designby-container', designData);
