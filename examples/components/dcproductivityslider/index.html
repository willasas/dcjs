使用说明

基本用法

1. 准备滑块项目数据数组，包含每个项目的标题、图片、描述等信息

2. 初始化DCProductivitySlider组件，传入配置选项

3. 组件会自动渲染滑块，并支持多种交互方式

滑块项目数据格式

每个项目对象包含以下属性：

- title: 项目标题

- backgroundImage: 背景图片路径

- thumbnail: 缩略图路径

- description: 项目描述

- buttonText: 按钮文本

- link: 按钮链接

- target: 链接打开方式（可选，默认为_self）

主要功能

• 卡片激活/失活动画效果

• 自动播放功能

• 导航按钮控制

• 分页点控制

• 键盘导航（左右箭头）

• 触摸滑动支持

• 响应式设计，适配移动端

• 链接跳转功能
