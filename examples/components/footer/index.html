Basic Usage

This example demonstrates the basic usage of the DCFooter component.

// Basic configuration for the footer component
const footerOptions = {
  container: 'main-footer',
  defaultTheme: 'light-theme',
  defaultLang: 'en-US',
  type: 'official',
  onThemeChange: (theme) => {
    console.log('Theme changed to:', theme);
  },
  onLangChange: (lang) => {
    console.log('Language changed to:', lang);
  }
};

// Initialize the footer component
const footer = new DC.Footer(footerOptions);

Custom Container Example

This example demonstrates how to use a custom container for the footer component.

// Custom container configuration
const customContainerOptions = {
  container: '#custom-footer-container',
  defaultTheme: 'light-theme',
  defaultLang: 'zh-CN',
  type: 'gaming'
};

// Initialize with custom container
const customFooter = new DC.Footer(customContainerOptions);

Different Footer Types

This example demonstrates the different footer types available.

// Different footer types

const gamingFooterOptions = {

const gamingFooter = new DC.Footer(gamingFooterOptions);

Dynamic Theme and Language Change

This example demonstrates how to dynamically change the theme and language.

// Initialize footer
const dynamicFooterOptions = {
  defaultTheme: 'light-theme',
  defaultLang: 'en-US',
  type: 'official'
};

const dynamicFooter = new DC.Footer(dynamicFooterOptions);

// Later, change theme
dynamicFooter.setTheme('dark-theme');

// Later, change language
dynamicFooter.setLanguage('zh-CN');
