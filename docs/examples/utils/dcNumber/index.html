dcNumber 工具类示例

1. format - 数字格式化

格式化数字，添加千分位分隔符，控制小数位数。

2. toChinese - 数字转中文大写金额

将数字转换为中文大写金额格式。

3. calculate - 数字精确计算

执行数字的精确计算，避免 JavaScript 浮点数精度问题。

4. clamp - 数字范围限制

将数字限制在指定范围内。

5. random - 生成随机数

生成指定范围内的随机数。

6. toFileSize - 字节转文件大小

将字节数转换为人类可读的文件大小格式。

7. toRoman - 数字转罗马数字

将数字转换为罗马数字格式（支持 1-3999）。

8. isPrime - 判断是否为质数

判断一个数字是否为质数。

9. factorial - 计算阶乘

计算数字的阶乘。

10. sqrt - 计算平方根

计算数字的平方根。

11. pad - 数字补零

为数字添加前导零，达到指定长度。

12. isInteger - 判断是否为整数

判断一个数字是否为整数。

13. percentage - 计算百分比

计算一个数字占总数的百分比。

14. round - 数字四舍五入

对数字进行四舍五入，保留指定小数位数。

15. 基本算术运算 - sum, subtract, multiply, divide

执行基本的算术运算。

16. count - 通用算术运算

使用指定运算符执行算术运算。
