dcUrl 工具类示例

1. parse - URL解析

解析 URL 字符串，返回 URL 组件对象。

2. parseQuery - 查询字符串解析

解析 URL 查询字符串，返回查询参数对象。

3. build - URL构建

根据组件构建完整的 URL 字符串。

4. getParam - 获取URL参数

从 URL 中获取指定的查询参数值。

5. updateParams - 更新URL参数

修改 URL 中的查询参数，返回新的 URL 字符串。

6. removeParams - 删除URL参数

从 URL 中删除指定的查询参数，返回新的 URL 字符串。

7. isValid - URL验证

检查 URL 字符串是否为有效的 URL 格式。

8. join - 路径合并

合并多个 URL 路径片段，返回合并后的路径。

9. getExtension - 获取文件扩展名

从 URL 中获取文件扩展名。

10. getDomain - 获取域名

从 URL 中提取域名。

11. isAbsolute - 绝对路径检查

检查 URL 字符串是否为绝对路径。

12. normalize - URL标准化

标准化 URL 字符串。

13. getRelativePath - 获取相对路径

获取从源路径到目标路径的相对路径。
