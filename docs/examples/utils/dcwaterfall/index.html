使用说明

  • 初始化瀑布流：创建 dcWaterfall 实例并传入容器选择器。
  • 加载更多：点击"加载更多"按钮可以动态添加新的瀑布流项。
  • 重置布局：点击"重置布局"按钮可以重新计算并布局所有瀑布流项。
  • 响应式布局：调整浏览器窗口大小，瀑布流会自动重新布局。

代码示例

// 初始化瀑布流
const waterfall = new dcWaterfall('#waterfall-container');

// 加载更多项
function loadMoreItems() {
  waterfall.dynamicLoadMoreItems(5);
}

// 重置布局
function resetLayout() {
  waterfall.resetLayout();
}
