dcObject 工具类示例

1. deepClone - 深拷贝对象

创建对象的深拷贝，包括嵌套对象、数组、日期和正则表达式。

2. merge - 合并对象

深度合并多个对象，支持嵌套对象和数组合并。

3. get/set/unset - 属性路径操作

根据路径获取、设置或删除对象的属性。

4. toQueryString/fromQueryString - 对象与查询字符串转换

在对象和 URL 查询字符串之间进行转换。

5. pick/omit/filter - 对象属性过滤

选择、排除或根据条件过滤对象的属性。

6. isEqual - 对象比较

深度比较两个对象是否相等。

7. flatten - 扁平化对象

将嵌套对象扁平化为单层对象，使用点号分隔路径。

8. rename - 对象属性重命名

根据映射关系重命名对象的属性。
