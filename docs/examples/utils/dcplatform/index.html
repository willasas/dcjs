DCJS - Platform 工具类示例

什么是 DCPlatform？

DCPlatform 是一个环境判断工具类，提供了丰富的方法用于检测当前运行环境的各种信息，包括：

  • 设备类型检测（PC、移动设备、平板、TV）
  • 操作系统检测（iOS、Android、Windows、Mac、Linux）
  • 浏览器类型检测（Chrome、Firefox、Safari、QQ浏览器等）
  • 应用环境检测（微信、QQ、微信小程序、支付宝小程序等）
  • 全屏模式检测
  • 内核检测（WebKit等）

设备类型检测

示例 2：检测设备类型

使用 DCPlatform 检测当前设备类型：

// 检测设备类型 const isPC = platform.isPC(); const isMobile = platform.isMobile(); const isTablet = platform.isTablet(); const isTV = platform.isTV(); // 获取设备类型 const deviceType = platform.getDeviceType(); // 返回 'mobile' | 'tablet' | 'pc' | 'tv' console.log('Is PC:', isPC); console.log('Is Mobile:', isMobile); console.log('Is Tablet:', isTablet); console.log('Is TV:', isTV); console.log('Device Type:', deviceType);

使用场景

场景 1：响应式布局适配

// 根据设备类型调整布局 if (platform.isMobile()) { // 移动端布局 document.body.classList.add('mobile-layout'); } else if (platform.isTablet()) { // 平板布局 document.body.classList.add('tablet-layout'); } else { // PC 布局 document.body.classList.add('pc-layout'); }

场景 2：浏览器兼容性处理

// 根据浏览器类型处理兼容性 if (platform.isIE()) { // IE 浏览器特殊处理 console.warn('您正在使用 IE 浏览器，部分功能可能受限'); } else if (platform.isSafari()) { // Safari 浏览器特殊处理 console.log('您正在使用 Safari 浏览器'); }

场景 3：应用环境适配

// 根据应用环境调整功能 if (platform.isWeChatMiniProgram()) { // 微信小程序环境 initWeChatMiniProgram(); } else if (platform.isWeChat()) { // 微信环境 initWeChatShare(); } else if (platform.isQQ()) { // QQ 环境 initQQShare(); }

场景 4：全屏模式检测

// 检测全屏模式并调整界面 if (platform.isFullscreen()) { // 全屏模式 document.getElementById('fullscreen提示').style.display = 'none'; } else { // 非全屏模式 document.getElementById('fullscreen提示').style.display = 'block'; }

浏览器兼容性

支持的浏览器

  • Chrome 4+
  • Firefox 3.5+
  • Safari 4+
  • Edge 12+
  • IE 9+
  • 移动端浏览器（Chrome、Safari、QQ浏览器、微信浏览器等）

注意：DCPlatform 工具类依赖于 navigator.userAgent 和 navigator.platform 属性，这些属性在不同浏览器中可能存在差异，因此检测结果可能会有一定的误差。

注意事项

1. 关于 userAgent 检测的局限性

DCPlatform 工具类主要基于 navigator.userAgent 进行检测，这种方法存在一定的局限性：

  • userAgent 可以被用户修改或伪装
  • 不同浏览器和设备的 userAgent 格式可能不同
  • 新设备和浏览器的 userAgent 可能未被覆盖

2. 关于检测结果的缓存

DCPlatform 工具类在初始化时会进行一次完整的检测，并将结果缓存起来。这意味着：

  • 大多数检测方法（如 isPC()、isMobile() 等）会返回缓存的结果，性能较好
  • isFullscreen() 方法会实时检测，因为全屏状态可能会动态变化

3. 关于小程序检测

小程序检测的准确性可能会受到平台更新的影响：

  • 微信小程序检测依赖于 window.__wxjs_environment 属性
  • 支付宝小程序和 QQ 小程序主要基于 userAgent 检测
