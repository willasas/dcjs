dprettylog 工具类示例

使用说明

dprettylog 是一个美化日志输出的工具类，提供了多种日志级别和特殊功能。

注意：所有日志输出都会显示在浏览器控制台中，请打开控制台查看结果。

按 F12 或右键选择「检查」打开浏览器开发者工具，切换到「控制台」标签页查看日志输出。

1. 基本日志级别

结果说明

点击上方按钮后，对应的日志会显示在浏览器控制台中，不同级别的日志会有不同的颜色。

2. 表格日志

点击下方按钮，会在控制台中以表格形式显示测试数据。

结果说明

控制台会显示一个格式化的表格，包含姓名、年龄和城市字段。

3. 图片日志

结果说明

点击按钮后，控制台会显示图片的详细信息和预览。

4. 环境信息

点击下方按钮，会在控制台中显示当前浏览器环境信息。

结果说明

控制台会显示当前浏览器名称、版本和用户代理字符串等信息。
