DCJS - QRCode 工具类示例

什么是 DCQRCode？

DCQRCode 是一个功能强大的二维码生成工具类，可以轻松生成各种样式的二维码。它具有以下特点：

  • 支持自定义二维码尺寸
  • 支持自定义背景色和前景色
  • 支持自定义边距
  • 支持不同的纠错级别（L、M、Q、H）
  • 可以直接渲染到指定元素
  • 返回 Data URL 格式的二维码图像
  • 支持编码各种类型的文本数据

基本用法

示例 1：生成基本二维码

使用默认配置生成二维码：

// 创建 QRCode 实例 const qrCode = new DC.QRCode(); // 生成二维码 const dataUrl = qrCode.generate('Hello, World!'); // 将二维码显示在页面上 const img = document.createElement('img'); img.src = dataUrl; document.getElementById('qr-container-1').appendChild(img);

示例 2：自定义二维码样式

使用自定义配置生成二维码：

// 创建 QRCode 实例 const qrCode = new DC.QRCode(); // 自定义选项 const options = { size: 250, // 二维码尺寸 bgColor: '#f0f0f0', // 背景色 fgColor: '#336699', // 前景色 margin: 8, // 边距 errorCorrectionLevel: 'H' // 纠错级别 }; // 生成二维码 const dataUrl = qrCode.generate('https://www.example.com', options); // 将二维码显示在页面上 const img = document.createElement('img'); img.src = dataUrl; document.getElementById('qr-container-2').appendChild(img);

示例 6：生成不同纠错级别的二维码

不同纠错级别的二维码可以在不同程度的损坏情况下仍然被识别：

// 创建 QRCode 实例 const qrCode = new DC.QRCode(); // 生成不同纠错级别的二维码 const errorLevels = ['L', 'M', 'Q', 'H']; const levelNames = { 'L': '低 (7%)', 'M': '中 (15%)', 'Q': '较高 (25%)', 'H': '高 (30%)' }; const text = 'https://www.example.com'; errorLevels.forEach(level => { const dataUrl = qrCode.generate(text, { size: 150, errorCorrectionLevel: level }); const qrItem = document.createElement('div'); qrItem.className = 'qr-item'; qrItem.innerHTML = `
纠错级别: ${levelNames[level]}
`; document.getElementById('qr-container-6').appendChild(qrItem); });

构造函数

方法

方法名 参数 返回值 说明
generate(text, options) text: string - 要编码的文本
options: object - 配置选项		 string - Data URL 格式的二维码图像 生成二维码并返回 Data URL
render(element, text, options) element: string|HTMLElement - 目标元素或元素 ID
text: string - 要编码的文本
options: object - 配置选项		 无 直接将二维码渲染到指定元素

选项配置

选项 类型 默认值 说明
size number 200 二维码尺寸（像素）
bgColor string #FFFFFF 背景色（CSS 颜色值）
fgColor string #000000 前景色（CSS 颜色值）
margin number 4 边距（模块数）
errorCorrectionLevel string M 纠错级别（L、M、Q、H）

使用场景

场景 1：网址链接

// 生成网址链接二维码 const qrCode = new DC.QRCode(); const dataUrl = qrCode.generate('https://www.example.com'); // 显示二维码 const img = document.createElement('img'); img.src = dataUrl; document.getElementById('qr-links').appendChild(img);

场景 2：联系信息

// 生成联系信息二维码（使用 vCard 格式） const vCard = `BEGIN:VCARD VERSION:3.0 FN:John Doe TEL:+1234567890 EMAIL:john@example.com URL:https://www.example.com END:VCARD`; const qrCode = new DC.QRCode(); const dataUrl = qrCode.generate(vCard); // 显示二维码 const img = document.createElement('img'); img.src = dataUrl; document.getElementById('qr-contact').appendChild(img);

场景 4：活动门票

// 生成活动门票二维码 const ticketInfo = JSON.stringify({ event: 'Tech Conference 2024', date: '2024-12-01', time: '10:00 AM', venue: 'Conference Center', ticketId: 'TC2024-001', attendee: 'John Doe' }); const qrCode = new DC.QRCode(); const dataUrl = qrCode.generate(ticketInfo, { size: 250, bgColor: '#f0f8ff', fgColor: '#000080', errorCorrectionLevel: 'H' }); // 显示二维码 const img = document.createElement('img'); img.src = dataUrl; document.getElementById('qr-ticket').appendChild(img);

浏览器兼容性

支持的浏览器

  • Chrome 4+
  • Firefox 3.5+
  • Safari 4+
  • Edge 12+
  • IE 9+

注意：DCQRCode 工具类依赖于 Canvas API，因此需要浏览器支持 Canvas 功能。在不支持 Canvas 的环境中，二维码生成将失败。

注意事项

1. 文本长度限制

二维码的容量有限，不同纠错级别的二维码可以容纳的文本长度不同：

  • L 级（7% 纠错）：约 152 个字符
  • M 级（15% 纠错）：约 128 个字符
  • Q 级（25% 纠错）：约 106 个字符
  • H 级（30% 纠错）：约 78 个字符

如果文本过长，二维码可能无法正确生成或难以扫描。

2. 颜色选择

为了确保二维码可以被正确扫描，建议遵循以下颜色选择原则：

  • 前景色应该比背景色深
  • 避免使用浅色作为前景色
  • 避免使用渐变或图案作为背景
  • 确保颜色对比度足够高

3. 尺寸选择

二维码的尺寸应该根据使用场景选择：

  • 网页显示：150-300 像素
  • 打印材料：至少 2x2 厘米
  • 远距离扫描：更大的尺寸

4. 边距设置

二维码周围应该有足够的空白边距，建议设置为 4 个模块大小（默认值），这样可以确保二维码可以被正确扫描。
