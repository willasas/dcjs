SimpleAdapt 工具类示例

说明：本示例展示了 SimpleAdapt 工具类的使用方法，该工具类用于页面适配，支持 rem 和 vw 两种适配方案。您可以通过下方的控制按钮切换适配单位，调整浏览器窗口大小查看适配效果。

工具类介绍

SimpleAdapt 是一个简单的页面适配工具，具有以下功能：

  • 支持 rem 和 vw 两种适配方案
  • 自动检测最接近的设计尺寸
  • 根据窗口大小自动调整缩放比例
  • 设置 CSS 变量，方便在样式中使用
  • 提供 px 转 rem、vw、vh 的方法
  • 支持移动设备检测
  • 防抖处理窗口 resize 事件，提高性能

控制选项

CSS 变量

SimpleAdapt 会设置以下 CSS 变量：

--scale: 1
--design-width: 1920px
--design-height: 1080px
--window-width: 1920px
--window-height: 1080px
--vw: 0.0520833333vw
--vh: 0.0925925926vh
--rem-base: 100px

使用示例

// 引入 SimpleAdapt 工具类
import SimpleAdapt from 'path/to/sampleadapt.js';

// 初始化适配工具
const simpleAdapt = new SimpleAdapt({
  unit: 'rem', // 适配单位：'rem' 或 'vw'
  rootFontSize: 100, // 基准字体大小
  designSize: {
    sizes: [
      { width: 1920, height: 1080 }, // PC端设计尺寸
      { width: 750, height: 1624 }, // 移动端设计尺寸
    ],
  },
});

// 切换适配单位
simpleAdapt.setUnit('vw');

// 设置新的设计尺寸
simpleAdapt.setDesignSizes([
  { width: 3840, height: 2160 }, // 4K 设计尺寸
  { width: 1920, height: 1080 }, // PC 设计尺寸
  { width: 750, height: 1624 }, // 移动端设计尺寸
]);

// 手动执行适配
simpleAdapt.adapt();

// 销毁适配实例
simpleAdapt.destroy();

// 单位转换
const remValue = simpleAdapt.px2rem(200); // 转换为 rem
const vwValue = simpleAdapt.px2vw(100); // 转换为 vw
const vhValue = simpleAdapt.px2vh(100); // 转换为 vh

CSS 使用示例

/* 使用 CSS 变量 */
.element {
  /* 使用 rem 单位 */
  width: 10rem;
  height: 10rem;

  /* 使用 vw 单位 */
  width: calc(100 * var(--vw));
  height: calc(100 * var(--vw));

  /* 使用缩放比例 */
  transform: scale(var(--scale));

  /* 使用设计尺寸 */
  max-width: var(--design-width);
  max-height: var(--design-height);
}
