DCVirtualCard 虚拟卡片组件示例

一个具有多种主题和交互功能的虚拟卡片组件

交互说明

  • 点击卡号： 复制卡号到剪贴板，会显示复制成功的提示
  • 双击卡片： 翻转卡片，显示卡片背面效果
  • 主题切换： 使用下方的主题按钮切换不同的卡片样式

使用方法

// 初始化 DCVirtualCard 组件 const card = new DC.VirtualCard({ virtualName: 'Virtual', virtualDesc: 'Virtual Card', logoUrl: 'https://assets.codepen.io/14762/airwallex-virtual.svg', cardNumber: '6230 5678 0012 3456 828', cardId: '00211', cardName: 'Mike Trilford', silkLogoUrl: 'https://assets.codepen.io/14762/visa-virtual.svg', theme: 'one', targetElement: document.getElementById('card-container') });
