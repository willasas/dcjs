DCSidenav 示例

Section 1

基本介绍

DCSidenav 是一个功能强大的侧边导航组件，用于实现内容区域的导航功能。

尝试向下滚动页面，当滚动超过200px时，侧边导航会自动显示。

这是第一个内容区域，包含了关于侧边导航组件的基本介绍。

侧边导航组件可以帮助用户快速导航到页面的不同部分，提高用户体验。

组件会根据页面滚动位置自动更新导航项的激活状态，并且支持点击导航项快速跳转到对应内容区域。

Section 2

功能特性

DCSidenav 组件支持以下特性：

  • 滚动显示/隐藏控制
  • 自动更新激活状态
  • 平滑滚动到对应区域
  • 自定义导航数据
  • 支持事件回调
  • 响应式设计

这是第二个内容区域，详细介绍了侧边导航组件的功能特性。

组件可以根据配置的阈值自动控制显示和隐藏，当页面滚动超过阈值时显示导航，否则隐藏导航。

当用户滚动页面时，组件会自动检测当前可见的内容区域，并更新对应导航项的激活状态。

Section 3

使用方法

使用 DCSidenav 组件非常简单，只需要以下几个步骤：

  1. 引入组件文件
  2. 创建导航数据
  3. 初始化组件
  4. 在内容区域添加 data-nav-section 属性

这是第三个内容区域，介绍了侧边导航组件的使用方法。

组件的初始化非常灵活，可以指定容器元素，也可以让组件自动创建容器。

导航数据支持自定义标题、链接和图标，用户可以根据需要配置导航项。

组件还支持点击回调，可以在用户点击导航项时执行自定义操作。

Section 4

API 参考

DCSidenav 组件提供了以下 API 方法：

  • addItem(item) - 添加导航项
  • setData(data) - 设置导航数据

这是第四个内容区域，提供了侧边导航组件的 API 参考。

通过 addItem 方法，可以动态添加新的导航项。

通过 setData 方法，可以批量设置导航数据，替换现有的导航项。

组件还支持多种配置选项，如滚动阈值、偏移量、激活状态类名等。
