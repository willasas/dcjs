DCSidebar 示例

基本用法

DCSidebar 是一个功能强大的侧边栏组件，支持以下特性：

  • 可折叠的侧边栏菜单
  • 自定义菜单项和图标
  • 返回顶部功能
  • 响应式设计
  • 平滑的动画效果
  • 丰富的API接口

API 参考

以下是 DCSidebar 组件的主要 API 方法：

  • setData(data) - 设置菜单数据
  • addMenuItem(item, index) - 添加菜单项
  • removeMenuItem(index) - 移除菜单项
  • updateMenuItem(index, newData) - 更新菜单项
  • expand() - 展开侧边栏
  • collapse() - 收起侧边栏
  • toggle() - 切换侧边栏状态
