DCVirtualScroller 虚拟滚动组件示例

组件介绍

DCVirtualScroller 是一个高性能的虚拟滚动组件，能够处理大量数据的高效渲染，适用于以下场景：

  • 长列表渲染（如商品列表、用户列表等）
  • 大数据量表格
  • 无限滚动加载
  • 任何需要高效渲染大量元素的场景

核心优势： 只渲染可视区域及其缓冲区的元素，大幅减少 DOM 节点数量，提升滚动性能。

1. 基本使用示例

最简单的虚拟滚动实现，使用默认渲染函数。

// 基本使用示例 const container = document.getElementById('basic-container'); const totalItems = 1000; const itemHeight = 60; const scroller = new DC.VirtualScroller( container, totalItems, itemHeight );

2. 自定义渲染函数示例

使用自定义渲染函数创建更复杂的列表项。

// 自定义渲染函数示例 const customContainer = document.getElementById('custom-container'); const customTotalItems = 500; const customItemHeight = 100; function customRenderItem(index) { const item = document.createElement('div'); item.className = 'custom-item'; item.style.height = `${customItemHeight}px`; item.style.position = 'absolute'; item.style.top = `${index * customItemHeight}px`; item.innerHTML = `
${index + 1}
商品 ${index + 1}
这是商品 ${index + 1} 的详细描述信息，包含商品的各种属性和特点。
`; return item; } const customScroller = new DC.VirtualScroller( customContainer, customTotalItems, customItemHeight, customRenderItem );
