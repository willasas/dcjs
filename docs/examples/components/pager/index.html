分页组件示例

基本用法

创建一个基本的分页组件，每页显示10个项目。

// 初始化分页组件
const pager1 = new DC.Pager({
    container: '#pager-container-1'
});
自定义每页显示数量

创建一个分页组件，每页显示5个项目。

// 初始化分页组件，自定义每页显示5个项目
const pager2 = new DC.Pager({
    container: '#pager-container-2',
    itemsPerPage: 5
});
自定义可见页码数量

创建一个分页组件，只显示5个可见页码。

// 初始化分页组件，自定义可见页码数量为5
const pager3 = new DC.Pager({
    container: '#pager-container-3',
    itemsPerPage: 3,
    visiblePages: 5
});
