DCTheme 主题切换示例

这是一个 DCTheme 组件的示例页面，展示了主题切换功能。您可以通过顶部的主题选择器选择不同的主题，页面会实时更新样式。

当前主题信息

当前激活的主题：light-theme

主题切换会触发回调函数，显示在控制台中。

主题效果展示

这个区域展示了不同主题下的样式效果：

  • 背景颜色：使用 var(--bg-theme-50)
  • 文本颜色：使用 var(--font-theme-900)
  • 边框颜色：使用 var(--color-primary-border)
  • 按钮背景：使用 var(--bg-theme-200)
