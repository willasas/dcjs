# 图标列表组件 (DCIconList)

## 简介

图标列表组件是一个用于在网页中展示多个图标的组件，支持以下功能：
- 按照类别组织图标
- 响应式布局，适配不同屏幕尺寸
- 点击图标可复制SVG代码到剪贴板
- 支持动态添加新的图标类别和图标
- 美观的悬停效果

## 目录结构

```
src/components/icons/
```

### 3. 初始化组件

使用准备好的图标数据初始化图标列表组件：

```javascript
// 初始化图标列表组件
const iconList = new DC.IconList('#icon-container', iconData);
```

参数说明：
- `container`：容器元素的 CSS 选择器，图标列表将被渲染到该容器中
- `content`：图标数据对象
- `options`：配置选项（可选）

## API 参考

### 构造函数

```javascript
new DCIconList(container, content, options);
```

### 方法

#### addIcon(category, icons)

向图标列表中添加新的图标类别或图标。

参数：
- `category`：图标的类别名称
- `icons`：要添加的图标数组，每个图标对象包含以下属性：
  - `name`：图标的名称
  - `description`：图标的描述
  - `url`：图标的 SVG 代码

示例：

```javascript
// 添加新图标
iconList.addIcon('newCategory', [
    {
        name: 'icon_new',
        description: '新图标',
        url: '<svg class="icon icon_new" width="64px" height="64px" viewBox="0 0 1024 1024"><path fill="#000000" d="M512 0C229.25 0 0 229.25 0 512s229.25 512 512 512 512-229.25 512-512S794.75 0 512 0z" /></svg>'
    }
]);
```

## 样式

组件会自动生成并插入以下样式：

- 响应式布局，适配不同屏幕尺寸
- 图标列表的网格布局
- 美观的悬停效果
- 清晰的图标类别标题

## 事件处理

- **点击事件**：点击图标时，会将图标的 SVG 代码复制到剪贴板，并显示提示信息

## 浏览器兼容性

- 支持所有现代浏览器（Chrome, Firefox, Safari, Edge）
- 不支持 IE11 及以下版本

## 常见问题

### 1. 图标列表不显示

**原因**：
- 容器元素不存在
- 图标数据格式不正确

**解决方案**：
- 确保容器元素的 CSS 选择器正确
- 检查图标数据格式是否符合要求

### 2. 点击图标后无法复制 SVG 代码

**原因**：
- 浏览器不支持 Clipboard API
- 页面没有用户交互

**解决方案**：
- 使用支持 Clipboard API 的现代浏览器
- 确保点击操作是由用户触发的

### 3. 响应式布局不生效

**原因**：
- CSS 媒体查询未正确应用

**解决方案**：
- 确保组件的 CSS 样式已正确加载
- 检查浏览器窗口大小是否符合媒体查询条件

## 总结

图标列表组件是一个功能完整、易于使用的组件，可用于展示和管理多个图标。它支持按类别组织图标、响应式布局、点击复制 SVG 代码等功能，适用于需要展示大量图标的场景，如图标库、设计系统文档等。
